Bayesian Analysis of Variance

The procedure calculates the Bayes factors and the posterior probabilities for a set of models using various combinations of the fixed and random independent variables as specified in the Models box. The posterior distributions for a single selected model can also be displayed. The models are numbered for reference in the Model Comparisons and Posterior Distribution fields, so it is useful to first run this procedure without using these numbers in order to see how the models are numbered.

The Bayes factor is the relative evidence in the data for one model over another. See the reference below for an introductory discussion.

Split files and the Statistics case weights are not supported by this procedure. Cases with missing values will be omitted from the analysis.

Dependent Specify the dependent variable.

Fixed Factors Specify the fixed factors. Only categorical (nominal and ordinal) variables can be used.

Random Factors Specify the random factors. Only categorical (nominal and ordinal) variables can be used.

Models This setting determines which models will be estimated from the set of factors.

  • All with main effects produces results for all combinations of the variables where the main effects for interaction terms are present.
  • Single Deletion produces all the models where exactly one factor is omitted.
  • Single Variable produces all the models that include a single factor.
  • Single Model with All Variables produces a single model containing all the independent variables. In this case, the Model Comparisons, Posterior Model Number, and settings on the Options subdialog do not apply, and the posterior distribution is always produced.
  • All produces results for all combinations of the variables. This may produce a very large number of models, so it should be used with caution.

    • Options

    Model Comparisons By default, the Bayes factors are computed relative to an intercept-only model plus any random factors. You can change the basis by specifying a model number. To see how the models are numbered run the specification once without using this option.

    Posterior Distribution You can display the posterior distribution of the model coefficients for a single model by specifying the model number.

    Omit random effects from posterior summary Check this box to omit the random effects terms from the posterior summary tables.

    Number of samples Specify the number of iterations (samples) to be drawn in calculating the posterior distribution.

    Plot model Bayes factors Check this box to display a graph of the Bayes factors. If the number of models is more than about ten, however, the chart will be very difficult to read.

    Maximum Number of Models Specify a limit on the number of models to be estimated.

    Prior Scale Fixed and Random Factors Choose the scales for the priors on all the standardized slopes. The scale parameters controls the expected true standardized effect size. For a particular scale 50% of the true effect sizes are within the interval. For "medium", this is (-.7071, .7071). Wide is (-1, 1), and ultrawide is (-1.414, 1.414).

    Bayes Factor Integration Iterations Some computations require numerical integration. You can set the number of iterations here.

    Workspace Action Choose whether to clear the workspace at the end of the procedure or retain the contents for further use in an R program in this session.

    Save Model Workspace Enter a file specification to save the model workspace to a file for future use.

    Additional Features

    This dialog generates syntax for the STATS BAYES ANOVA extension command. To display syntax help for this command, run the following syntax:

    STATS BAYES ANOVA /HELP.

    In Statistics version 23 or later, you can place the cursor within the command in the Syntax Editor and press F1.

    Using syntax you can specify numerical prior values.
